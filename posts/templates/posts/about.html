-> Privalise is an Open-Source Social Media network that focuses on the privacy and security of it's users. it's made by and for the community with no profits Or ads at all

-> Well, Privalise doesn't record any data at all about it's users (No ips, no emails/phone numbers, no tracking or any fingerprinting). So the only data we have on you is what you actually post, comment, etc... hence even if our database was leaked no one would be able to identify you. and sensitive services like messages, notes, and the password manager are end-to-end encrypted meaning no one can read your them except you

-> Everything Privalise uses is Open-Source. We do not use any api or remote resources, meaning that any framework we use like bootstrap, fontawesome, django, etc... is stored locally to terminate any possibilty of user-tracking on the website. What we specifically use:
  • Python 3
  • Bootstrap 5
  • Django 4
  • Html/Css
  • FontAwesome 5
